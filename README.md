# 用vuejs仿网易云音乐
***
在做完前两个基础的静态页面后，复习了一遍css基础决定写一个非静态页面来巩固之前学的vue全家桶，花了差不多半个月
时间来大概完成这个网易云小项目。
## 技术栈
***
* vue+vue-router(核心)
* axios(处理请求)
* vuex(全局数据流传输)
* bootstrap(响应式框架)
* animate(处理简单动画效果)
## 功能分析与设计
***
为了尽可能的与官网相同，采用的GitHub上已经写好的后端api当作此项目的请求接口
